# GreenCubic3D
 The GreenCubic, but made with Unity in 3D!

## Controls
 WASD - Walk
 
 Space - Jump
 
 Shift - Sprint
 
 R - Reload level
 
 Esc - Exit Game
 
 Mouse - Rotate

 Right Mouse Button - Use (Example: Open door)

 Hold and drag Right Mouse Button - rotate camera.
