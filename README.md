# GreenCubic3D
 The GreenCubic, but made with Unity in 3D!

## Controls
 WASD - Walk
 
 Space - Jump
 
 Shift - Sprint
 
 R - Reload weapon
 
 Esc - Exit Game
 
 Mouse - Rotate

 Right Mouse Button - Use (Example: Open door)

 Left Mouse Button - Attack

 E - grab object
